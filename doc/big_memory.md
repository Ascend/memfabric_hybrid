### Big Memory 

#### Feature list
* Basic features:
  * Create unified memory space to store data
  * Data operations in sync and async way
  * Sync data operation includes: L2G, G2L, G2H, H2G
  * Async data operation (TODO)
  * Auto ranking
  * Dynamically join and leave
  * Heartbeat (TODO)
  * Languages supported: c, python
  
Note:
```angular2html
L2G: copy data from local space to global space
G2L: copy data from global space to local space
H2G: copy data from host memory to global space
G2H: copy data from global space to host memory
```
